# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-08

### Added

- Initial release with three classic arcade games
- Brickrush game with 8 unique levels
- 1024 puzzle game with tile merging mechanics
- Snake game with bonus food system
- Game selector with keyboard shortcuts (1, 2, 3)
- Enter key shortcut to start games from menu
- Responsive design for desktop, tablet, and mobile
- Touch controls for mobile devices
- High score persistence using localStorage
- Power-up system for Brickrush (Multiball, Stretch Paddle)
- How to Play modals for each game
- Pause functionality for all games
- Game over and win screens with statistics
- Dark theme interface
- Smooth animations and visual effects
- Loading screen with animated elements

### Technical

- Built with React 18 and functional components
- Vite for fast development and optimized builds
- Tailwind CSS for responsive styling
- HTML5 Canvas for game rendering
- Custom React hooks for game logic
- Mobile-first responsive approach
